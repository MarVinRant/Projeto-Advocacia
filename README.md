# ⚖️ Escritório de Advocacia & Consultoria Jurídica - Landing Page

## 📖 Sobre o Projeto
Uma landing page sóbria, elegante e focada em conversão, desenvolvida para escritórios de advocacia, advogados autônomos e consultorias jurídicas. O objetivo principal deste projeto é transmitir autoridade, confiança e segurança, criando um ambiente digital profissional que facilita a captação de clientes e o agendamento de consultas jurídicas.

Este projeto faz parte do meu portfólio profissional de desenvolvimento Front-End e demonstra minha capacidade de criar soluções digitais para o setor jurídico, onde a clareza das informações e o design focado em credibilidade são essenciais para fechar contratos.

## ✨ Principais Funcionalidades
* **Design Sóbrio e Confiável:** Uso de uma paleta de cores tradicional (como tons de azul-marinho, preto, branco e dourado) e tipografia elegante para refletir a seriedade e o rigor que a profissão exige.
* **Mobile-First para Urgências:** Como muitas buscas por advogados ocorrem em momentos de necessidade imediata através do celular, a interface foi rigorosamente otimizada para ser rápida e acessível em qualquer dispositivo.
* **Áreas de Atuação:** Seções claras e objetivas para apresentar as especialidades do escritório (Direito Civil, Trabalhista, Criminal, Empresarial, etc.), ajudando o cliente a identificar rapidamente se o escritório resolve o problema dele.
* **Agendamento e Triagem (CTA):** Botões de "Falar com um Advogado" ou "Agendar Consulta" estrategicamente posicionados, guiando o cliente diretamente para o WhatsApp ou formulário de triagem.
* **Performance Otimizada:** Código limpo e carregamento rápido para garantir que o cliente não desista de entrar em contato devido à lentidão.

## 🛠️ Tecnologias Utilizadas
A página foi construída "na unha", garantindo controle absoluto sobre a estrutura, a segurança e a velocidade de carregamento, sem a necessidade de frameworks externos pesados:

* **HTML5:** Estrutura semântica focada em um excelente ranqueamento orgânico (SEO local), fundamental para o escritório ser encontrado por clientes da região no Google.
* **CSS3:** Estilização responsiva, utilizando Grid e Flexbox para organizar os blocos de texto, áreas de atuação e o perfil dos sócios com alinhamento perfeito.
* **JavaScript (Vanilla):** Implementação de interatividade pontual, focada em melhorar a experiência do usuário (UX), como botões dinâmicos e seções de dúvidas frequentes (FAQ).
